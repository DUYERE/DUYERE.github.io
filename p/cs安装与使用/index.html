介绍

是一款以Metasploit为基础的GUI的框架式渗透工具，集成了端口转发、服务扫描、自动化溢出、多模式端口监听、Winexe木马生成、office宏病毒生成、木马捆绑等；钓鱼攻击包括：站点克隆、目标信息获取、Java执行、浏览器自动攻击等。

实际上 Cobaltstrike 的社区版是大家熟知的 Armitage (一个 MSF 的图形化界面工具)，而 Cobaltstrike 大家可以理解其为 Armitage 的商业版。

安装

在vps上安装好cs，并将权限给到位，执行以下命令

cd Server/
chmod +x teamserver
chmod +x TeamServerImage
./teamserver ip 密码

如下图：

下图是正确执行后的样子

然后在本机打开client客户端，输入刚才填写的ip以及密码

然后就可以正常运行了

上线

设置监听器

点击“监听器”

点击“add”，然后配置好ip及端口，保存即可

生成木马后门

如图点击“Windows可执行程序”

点击框选的那个选择框，直接选择刚才配置的监听器

保存至想上传的位置

在桌面生成的exe文件，双击运行即可上线

下图可以看到就是连接上了

默认60秒执行命令，用命令sleep 3 设置为全部命令3秒执行

还有就是执行系统命令时要在前面加shell，如shell whoami

插件加载

如图，依次点击

可以看到多了“谢公子的插件”这个选项，插件已经成功加载

联动MSF

CS->MSF (CS 把权限转交给 MSF)

CS：
新建个监听器，监听的协议要与 MSF 一致，如CS监听 HTTP 协议，那么 MSF 也要监听 HTTP 协议
CS 监听的协议是 windows/foreign/reverse_http 字样

然后就执行spawn msf 这个命令就可以联动了
